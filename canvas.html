HTML5 Canvas 2D Island Generator

This was a small project to look at the HTML5 Canvas element and practise some JavaScript.

In the top box is a randomly generated 'island', a cell can be deep water, shallow water, sand, grass, a tree or rock. The outer 3 rows/columns are always deep water, the middle is randomly generated. Use this button below to re-generate:

Left click a cell in the top box to change the cell type.

Your browser does not support the canvas element.

The middle box is a zoomed-in view of the island, the pink cell is a character. The character can walk on certain cells (not deep water, trees or rock) by using the ⬅, ⬆, ⬇ and ➡ keys. When the island is generated the character is positioned on a random 'walkable' cell.

Your browser does not support the canvas element.

The bottom box is a console, just seemed like a handy way to output information (I do know there's a JavaScript console 😉)
